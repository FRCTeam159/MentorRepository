NI Software for FRC 2015 Readme

September 2014

The NI Software for FRC 2015 includes the LabVIEW 2014 Professional Development System and other software components that you can use to program your robot. To obtain the FRC API, examples, and utilities like the Driver Station and roboRIO Imaging Tool, install the FRC 2015 Update Suite after the FRC kickoff. Visit the National Instruments website for more information about the software updates after kickoff.


Installation Options

NI Software for FRC 2015 includes two installation options. Select the option that applies to your development needs.


Option 1: Install Everything for LabVIEW Development

This option allows you to program your robot in LabVIEW, perform vision analysis, and capture from USB cameras. This option installs the following items:

  • LabVIEW 2014 Professional Development System
  • NI Vision Development Module
  • NI-IMAQdx driver
  • Other necessary add-ons for programming in the LabVIEW environment
Note: This option does not install the FRC API, examples, or utilities like the Driver Station and roboRIO Imaging Tool. Install the FRC 2015 Update Suite after the FRC kickoff to obtain these features.

Option 1 has the following system requirements:

  • Windows 8/7
  • 4.62 GB of disk space


Option 2: Install Only NI VIsion Development Module

This option allows you to perform vision analysis and capture from USB cameras if you program your robot outside of LabVIEW. This options installs the following items:

  • NI Vision Development Module
  • NI-IMAQdx driver

Option 2 has the following system requirements:

  • Windows 8/7
  • 2.80 GB of disk space
